post  

Creacion de un nuevo estudiante del Proyecto Nahual

  • Request
  • Response

Query Parameters

  • name required (string)

    Nombre del estudiante

    Example: Daniel

  • lastName required (string)

    Apellido del estudiante

    Example: Apellido

  • node required (number)

    Id de un nodo

  • graduationDate (date)

    Example: 2018-03-28

  • courseDate (date)

    Example: 2018-03-28

  • email (string)

    Email del estudiante

    Example: estudiante@gmail.com

  • phone (string)

    Teléfono del estudiante

    Example: 1112223333

  • resumeUrl (string)

    URL del CV del estudiante

    Example: linkedin.com/estudiante

  • working required (boolean)

    True si el estudiante está trabajando

    Example: True

  • lookingForWork required (boolean)

    True si el estudiante está buscando trabajando

    Example: True

  • feedback (string)

    Feedback del profesor

    Example: Feedback
