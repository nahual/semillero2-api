graduated

graduados del Proyecto Nahual

/graduated get post

get

Devuelve una coleccion de Graduados

post

Creacion de un nuevo graduado del Proyecto Nahual

get /graduated

Devuelve una coleccion de Graduados

  • Request
  • Response

Query Parameters

  • año: (number - default: -1)
  • mes: (number - default: -1)
  • dia: (number - default: -1)
  • nodeId: (number - default: -1)

HTTP status code 200

Obtencion exitosa de todos los graduados(Incluso los que estan marcados con borrado logico)

Body

Type: application/json

Example:

[
  {
    "id": 1,
    "name": "santi",
    "date": 1481227669000,
    "deleted": true,
    "node": null
  },
  {
    "id": 2,
    "name": "Daniel",
    "date": 1481305396000,
    "deleted": false,
    "node": {
      "id": 1,
      "name": "Banfield",
      "address": "CalleFalsa123"
    }
  }
]

post /graduated

Creacion de un nuevo graduado del Proyecto Nahual

  • Request
  • Response

Query Parameters

  • name: required (string)

    Nombre del graduado

    Example:

    Daniel
  • nodeId: required (number)

    Id de un nodo

HTTP status code 200

Creacion exitosa de un graduado

Body

Type: application/json

Example:

{
  "id": 2,
  "name": "Daniel",
  "date": 1481305395815,
  "deleted": false,
  "interviews": [],
  "node": {
    "id": 1,
    "name": "Banfield",
    "address": "CalleFalsa123",
    "graduateds": [
      2
    ]
  }
}

HTTP status code 400

Error en la creacion de un graduado por falta de parametros

Body

Type: application/json

Example:

{
  "timestamp": 1481306059861,
  "status": 400,
  "error": "Bad Request",
  "exception": "org.springframework.web.bind.MissingServletRequestParameterException",
  "message": "Required String parameter 'name' is not present",
  "path": "/graduated"
}
